# MegaBIT
Start to mine now!

## Prerequisites
Node.js and npm

## Getting Start

Clone this repo into new project folder (e.g., `my-proj`).
```shell
git clone https://github.com/Boffka/megabit.git  megabit
cd megabit
npm run apisetup
npm run websetup
npm run apistart
npm run webstart
```
open your browser on [http://localhost:4200/](http://localhost:4200/)
